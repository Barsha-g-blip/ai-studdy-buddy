An AI-powered Study Assistant built using Flask and Groq API.
This application allows users to ask study-related questions and receive intelligent responses in real-time.

 Features

💬 Ask questions and get AI-generated answers

⚡ Fast responses using Groq LLM

🧠 Uses Llama 3.3 70B model

🌐 Simple and clean web interface

🔐 Secure API key using environment variables

🛠️ Tech Stack

Python

Flask

Groq API

HTML / CSS

JavaScript (for frontend interaction)
